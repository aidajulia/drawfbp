Initial Information Packets (IIPs)

These are shown on the diagram as long, shallow rectangles with rounded corners.  The length of the rectangle is adjusted to match the length of the contents.

Right clicking on an IIP brings up a popup menu showing operations that can be performed on it.These are the following:

  • "Edit Item"
  • Toggle Visible/Invisible
  • "Delete"

An IIP can be connected to a process by an arrow - see the section on Arrows - but the arrow will be displayed without an arrowhead. This type of connection can be drawn in either direction - however, if it is drawn in the wrong direction, DrawFBP will try to swap the ends, which may give strange results if there are bends in the arrow.
